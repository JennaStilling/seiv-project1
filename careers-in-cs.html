Careers in Computer Science

What Careers Can You Expect to Find?

There are a wide variety of careers in the career science field. Whether you are wanting to go into the cyber defense field, game development, software engineering, or data analysis, there is a spot for you.

Note: The average salaries below are across the U.S.A. and may not reflect geographical and cost of living differences.


Software Engineer

Average Starting Salary: $64,000

Job Description:

Designs, develops, and maintains software applications by writing code, debugging, and collaborating with cross-functional teams to deliver reliable solutions for users

Information Security Analyst

Average Starting Salary: $62,000

Job Description:

Protects an organization’s systems and networks by monitoring, detecting, and responding to cyber threats, ensuring data confidentiality, integrity, and availability

Game Programmer

Average Starting Salary: $65,000

Job Description:

Develops the code that drives video games, working closely with designers and artists to implement gameplay mechanics, graphics, AI, and physics

Web Developer

Average Starting Salary: $55,000

Job Description:

Creates and maintains websites and web applications, focusing on the front-end (user interface) or back-end (server-side logic), ensuring functionality, performance, and responsiveness across different devices

Cloud Engineer

Average Starting Salary: $70,000

Job Description:

Designs, implements, and manages cloud-based infrastructure and services, ensuring scalability, security, and efficiency of cloud environments for organizations using platforms like AWS, Azure, or Google Cloud

Graphics Programmer

Average Starting Salary: $65,000

Job Description:

Specializes in writing code for rendering graphics in video games or other visual media, optimizing performance, and creating visually immersive environments, often working with tools like DirectX or OpenGL

Data Scientist

Average Starting Salary: $63,000

Job Description:

Analyzes large datasets to extract insights and solve complex problems using statistical methods, machine learning, and data visualization, helping organizations make data-driven decisions​

Mobile Application Developer

Average Starting Salary: $61,000

Job Description:

Builds and maintains applications for mobile devices, such as smartphones and tablets, using programming languages like Swift, Kotlin, or Java, ensuring functionality, performance, and user experience​
